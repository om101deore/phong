
# Phong Model

implementation of phong model using opengl, c++ and glsl

### Video
[![Youtube](https://www.youtube.com/watch?v=A1S9zvXsRC8)] 

### Dependencies
1. glad (opengl implementation)
2. glfw (window and input handeling)
3. glm  (math library)
4. stb  (image loader)

### Execute
use cmake

after cloning
``` bash
$ cd phong
$ mkdir bin
$ cd bin
$ cmake ../src
```
this will generate makefiles

to build and run
``` bash
$ make
$ ./run
```
